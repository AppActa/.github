# Convenção de Repositórios

## Regras gerais

- Utilizar apenas letras minúsculas
- Utilizar hífen para separar palavras
- Não utilizar espaços
- Não utilizar acentos
- Não utilizar caracteres especiais

## Estruturas

### Exemplos válidos

- `acta-web`
- `acta-mobile`
- `acta-pg-api`

### Exemplos inválidos

- `ACTA_API`
- `acta mobile`
- `DatabaseActa`
